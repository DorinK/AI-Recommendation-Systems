		  	    ex3.py  	

  		  ** Running Instructions **           	        

+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-


In the current working directory you should have:
1. ex3.py
2. books.csv
3. ratings.csv
4. users.csv
5. test.csv
4. requirements.txt


First, please run:	pip install -r requirements.txt
And then:	 	python3 ex3.py


The program does not take any parameters.
The program has an optional function of 'run_all' that prints all the recommendation tables and stats - if you want to enable this option then uncomment the 'run_all' command at the end of the file and then run the program.
